# Job4j Cinema

Веб-приложение для покупки билетов в кинотеатр.

## Функционал

- Регистрация и авторизация пользователей
- Просмотр расписания киносеансов
- Просмотр списка фильмов
- Покупка билетов
- Защита от покупки уже занятого места
- Отображение результата успешной/неуспешной покупки

## Технологии

- Java 21
- Spring Boot
- Spring MVC
- Thymeleaf
- Bootstrap 5
- PostgreSQL
- Liquibase
- Sql2o
- Maven

## Архитектура

Проект построен по слоистой архитектуре:

- Controller
- Service
- Repository
- Model / DTO

## База данных

Для работы используется PostgreSQL.

Liquibase автоматически создает схему БД и заполняет тестовые данные.

## Запуск проекта

1. Создать БД `cinema`
2. Настроить `application.properties`
3. Запустить приложение:

```bash
mvn spring-boot:run
```

## Возможности

### Гость
- Просмотр фильмов
- Просмотр расписания

### Авторизованный пользователь
- Покупка билетов
- Просмотр доступных мест
